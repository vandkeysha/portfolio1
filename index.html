MY SKILLS

HTML

HTML (HyperText Markup Language) is a standard markup language used to structure web pages. HTML provides the basic framework of a website by using elements such as text, images, tables, links, forms, and so on.

CSS

CSS (Cascading Style Sheets) is a language used to organize the appearance and layout of elements on web pages created with HTML. With CSS, you can change the color, size, margin, padding, position, animation, and more, so that the web page looks attractive and structured.

Javascript

JavaScript is a programming language used to make web pages interactive and dynamic. JavaScript runs in the user's browser (client-side) and allows developers to add features such as animation form validation, visual effects, to complex web applications.

PHP

PHP (Hypertext Preprocessor) is a server-side programming language used to build dynamic web pages. PHP runs on the server and generates content that is sent to the user's browser, such as HTML, CSS, or JSON.
